Brewtique

Ecommerce site for boutique beer, wine, and cider

Brewtique is an ecommerce platform featuring boutique beer, wine, and ciders. Upon entering the site, users are directed to the main browsing page where they can search by category, add products to their cart, and pay for purchases. Note: credit card submissions are sandboxed and cards are not actually charged.

Admin may also scroll down on the greeting page to log into the admin terminal. Admin terminal allows admin to edit information of each product, and also add new products using AWS S3 uploads.

Start browsing and purchasing brews on Brewtique.

Cook Book Game

Earn points by cooking and unlocking higher level recipes

Cook Book game that teaches users how to cook. Users can browse through a list of recipes, add recipes to their watch list, and earn points when they cook recipes. A user starts off as a starving college student with access to all level 1 recipes. As the user earns points he/she then becomes a chef's apprentice and unlocks all level 2 recipes. When a user reaches level 5, a nevigation tab becomes availables and lets these Executive chef level users add recipes to the platform.
